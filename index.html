Propuesta de Solución

Mi propuesta es la siguiente. Elaborar un servicio RESTful con una webapp como puntos de acceso.

La lógica del backend se administraría con firebase, lo cual nos permitiría escalar fácilmente al desarrollo de otros puntos de acceso, como una app. Aprovechando el servicio de cloud messaging se podrían mandar los monitoreos a un usuario en específico o a un grupo de ellos.

El porcenaje de monitoreos recibidos se podría almacenar en una base de datos para realizar análisis y generar estadisticas. Estas estadísticas podrían ser elaboradas automaticamente cada cierto periodo de tiempo. .
